# Game Design Doc


## Objective

- Game type: Isometric RPG
- Art Style: Low-Poly Style
- Game map: Grid-based movement


## Features

- Procedural dungeons
- Turn-based combat
- Enemies/monsters (AI)
- Inventory/loot
- Weapons (procedural?)
- Quests
- NPCs
- Town
- Charactter stats


## Development process

- Start simple
- Build layer by layer
- Focus on gameplay first, graphics and
polish and optimization come later.
